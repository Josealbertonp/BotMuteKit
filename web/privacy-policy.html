Política de Privacidade

Bot Monitor de Áudio Discord

Última atualização: 14 de agosto de 2025
Resumo Rápido: Este bot monitora apenas o status de áudio em canais de voz. Não coletamos dados pessoais, não armazenamos conversas e não compartilhamos informações com terceiros.

1. Informações que Coletamos

1.1 Dados Processados Temporariamente

Tipo de Dado Finalidade Duração
Status de áudio (mudo/não mudo) Detectar quando mover usuários Tempo real apenas
Canal de voz atual Saber onde o usuário está Tempo real apenas
ID do usuário Discord Gerenciar timers internos Enquanto timer ativo
Nome do usuário Logs operacionais Até limpeza automática

1.2 Dados que NÃO Coletamos

  • ❌ Conteúdo de mensagens de texto
  • ❌ Áudio de conversas
  • ❌ Dados pessoais (email, telefone, endereço)
  • ❌ Histórico de navegação
  • ❌ Informações de outros aplicativos
  • ❌ Dados de pagamento
  • ❌ Localização geográfica

2. Como Usamos as Informações

2.1 Finalidades do Processamento

  • Funcionalidade Principal: Monitorar status de áudio para mover usuários automaticamente
  • Logs Operacionais: Registrar ações para depuração e melhoria do serviço
  • Gerenciamento de Timers: Controlar tempo de espera antes de mover usuários

2.2 Base Legal (LGPD/GDPR)

  • Legítimo Interesse: Automação de moderação de canais de voz
  • Consentimento: Ao adicionar o bot ao servidor
  • Execução de Contrato: Fornecimento do serviço solicitado

3. Armazenamento e Retenção

Política de Não Armazenamento: Todos os dados são processados em tempo real. Nenhuma informação pessoal é armazenada permanentemente.

3.1 Dados Temporários

  • IDs de usuários: Removidos automaticamente quando timer expira ou usuário sai do canal
  • Logs operacionais: Limpos automaticamente após período configurado
  • Status de voz: Processado instantaneamente, não armazenado

3.2 Localização dos Dados

  • Processamento ocorre na infraestrutura do Replit
  • Dados transitam apenas entre Discord e nosso bot
  • Nenhum banco de dados externo é utilizado

4. Compartilhamento de Dados

Política de Não Compartilhamento: Não compartilhamos, vendemos ou transferimos dados para terceiros.

4.1 Exceções (apenas se legalmente obrigatório)

  • Ordem judicial válida
  • Investigação de crimes graves
  • Proteção de direitos fundamentais

5. Seus Direitos

5.1 Direitos LGPD/GDPR

  • Acesso: Solicitar informações sobre dados processados
  • Correção: Corrigir dados incorretos (limitado pela natureza do bot)
  • Exclusão: Remover o bot do servidor para parar o processamento
  • Portabilidade: Não aplicável (não há dados armazenados)
  • Oposição: Deixar canais monitorados ou remover o bot

5.2 Como Exercer seus Direitos

  • Parar o monitoramento: Sair do canal de voz ou ativar áudio
  • Parar completamente: Administrador remove o bot do servidor
  • Solicitar informações: Contatar através dos canais oficiais

6. Segurança

6.1 Medidas de Proteção

  • Criptografia em trânsito (HTTPS/WSS)
  • Acesso limitado apenas às permissões necessárias
  • Código aberto para auditoria pública
  • Processamento local sem transmissão externa

6.2 Limitações de Segurança

  • Dependemos da segurança da plataforma Discord
  • Administradores do servidor têm controle sobre configurações
  • Logs podem conter nomes de usuários em texto claro

7. Menores de Idade

Este bot não coleta intencionalmente dados de menores de 13 anos. Se um menor usar Discord e estiver em canais monitorados, apenas o status de voz será processado conforme descrito nesta política.

8. Transferências Internacionais

Os dados podem transitar por servidores Discord em várias jurisdições, mas seguem sempre esta política de privacidade e as proteções aplicáveis.

9. Alterações nesta Política

  • Mudanças serão comunicadas através dos canais oficiais
  • Alterações significativas terão período de adaptação
  • Uso continuado constitui aceitação das mudanças

10. Conformidade Legal

Esta política está em conformidade com:

  • Lei Geral de Proteção de Dados (LGPD) - Brasil
  • General Data Protection Regulation (GDPR) - União Europeia
  • Política de Privacidade do Discord
  • Diretrizes de Desenvolvedor do Discord

11. Contato - Encarregado de Dados

Para questões sobre privacidade e proteção de dados:

  • Canal principal: Issues no repositório GitHub
  • Tempo de resposta: Até 30 dias úteis
  • Idiomas: Português e Inglês
