Privacy Policy

Discord Audio Monitor Bot

Last updated: August 14, 2025
Quick Summary: This bot only monitors audio status in voice channels. We don't collect personal data, don't store conversations, and don't share information with third parties.

1. Information We Collect

1.1 Temporarily Processed Data

Data Type Purpose Duration
Audio status (muted/unmuted) Detect when to move users Real-time only
Current voice channel Know where user is located Real-time only
Discord user ID Manage internal timers While timer is active
Username Operational logs Until automatic cleanup

1.2 Data We DO NOT Collect

  • ❌ Text message content
  • ❌ Audio conversations
  • ❌ Personal data (email, phone, address)
  • ❌ Browsing history
  • ❌ Information from other applications
  • ❌ Payment data
  • ❌ Geographic location

2. How We Use Information

2.1 Processing Purposes

  • Core Functionality: Monitor audio status to automatically move users
  • Operational Logs: Record actions for debugging and service improvement
  • Timer Management: Control waiting time before moving users

2.2 Legal Basis (GDPR)

  • Legitimate Interest: Voice channel moderation automation
  • Consent: By adding the bot to server
  • Contract Performance: Providing requested service

3. Storage and Retention

No Storage Policy: All data is processed in real-time. No personal information is permanently stored.

3.1 Temporary Data

  • User IDs: Automatically removed when timer expires or user leaves channel
  • Operational logs: Automatically cleaned after configured period
  • Voice status: Processed instantly, not stored

3.2 Data Location

  • Processing occurs on Replit infrastructure
  • Data transits only between Discord and our bot
  • No external databases are used

4. Data Sharing

No Sharing Policy: We do not share, sell, or transfer data to third parties.

4.1 Exceptions (only if legally required)

  • Valid court order
  • Investigation of serious crimes
  • Protection of fundamental rights

5. Your Rights

5.1 GDPR Rights

  • Access: Request information about processed data
  • Rectification: Correct incorrect data (limited by bot nature)
  • Erasure: Remove bot from server to stop processing
  • Portability: Not applicable (no data stored)
  • Objection: Leave monitored channels or remove bot

5.2 How to Exercise Your Rights

  • Stop monitoring: Leave voice channel or activate audio
  • Stop completely: Administrator removes bot from server
  • Request information: Contact through official channels

6. Security

6.1 Protection Measures

  • Encryption in transit (HTTPS/WSS)
  • Access limited to necessary permissions only
  • Open source code for public audit
  • Local processing without external transmission

6.2 Security Limitations

  • We depend on Discord platform security
  • Server administrators control settings
  • Logs may contain usernames in plain text

7. Children's Privacy

This bot does not intentionally collect data from children under 13. If a minor uses Discord and is in monitored channels, only voice status will be processed as described in this policy.

8. International Transfers

Data may transit through Discord servers in various jurisdictions but always follows this privacy policy and applicable protections.

9. Changes to This Policy

  • Changes will be communicated through official channels
  • Significant changes will have adaptation period
  • Continued use constitutes acceptance of changes

10. Legal Compliance

This policy complies with:

  • General Data Protection Regulation (GDPR) - European Union
  • California Consumer Privacy Act (CCPA) - California, US
  • Discord Privacy Policy
  • Discord Developer Guidelines

11. Contact - Data Protection Officer

For privacy and data protection questions:

  • Primary channel: Issues on GitHub repository
  • Response time: Up to 30 business days
  • Languages: English and Portuguese
