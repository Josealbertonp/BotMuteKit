2. Service Description

The Audio Monitor is an automated bot that:

  • Monitors voice channel audio status of users
  • Automatically moves users with deactivated audio to a designated channel
  • Operates silently without commands or direct interactions
  • Allows configuration of monitored channels and timeout periods

3. Data Collection

The bot only processes:

  • User voice status (muted/unmuted, current channel)
  • User and channel names for operational logs
  • User IDs temporarily for timer management

We do NOT collect or store:

  • Text messages or audio content
  • Personal data beyond what's necessary for operation
  • Information from other bots or applications

4. Privacy and Security

  • All data is processed in real-time and not permanently stored
  • Operational logs are kept only for debugging and automatically cleaned
  • The bot does not access information outside configured voice channels
  • No data is shared with third parties

5. Required Permissions

The bot requires the following permissions to function:

  • View Channels: To identify voice channels

  • Manage Channels: To create AFK channel if needed
